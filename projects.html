Multi-Modal Fashion Discovery Engine
View on GitHub

  • Developed an end-to-end hybrid recommendation system for the H&M Personalized Fashion dataset, processing over 31 million transactions.
  • Engineered a high-performance data pipeline using Polars and Parquet, reducing memory overhead by 70% and accelerating ETL speed by 10x compared to standard Pandas workflows.
  • Implemented a Deep Learning feature extraction pipeline using a pre-trained ResNet50 (CNN) in PyTorch to generate 2048-dimensional embeddings from product images.
  • Developed a multi-stage retrieval strategy combining Collaborative Filtering (user history), Semantic Matching (metadata), and Visual Similarity (Computer Vision).
  • Achieved a 3x improvement in MAP@12 (Mean Average Precision) over the global popularity baseline through rigorous temporal validation and hybrid model tuning.
  • Built and deployed an interactive Streamlit dashboard allowing users to perform real-time visual discovery and navigate recommendations through an intuitive web interface.
  • Optimized system performance for local execution on consumer hardware by implementing resource caching and vectorized similarity calculations.

Biomechanical Injury Detection
View Project Website

  • Designed an end-to-end machine learning pipeline to analyze human movement and predict biomechanical stress from short exercise videos.
  • Used YOLOv8-based pose estimation to extract joint keypoints frame-by-frame, converting raw video into structured motion data.
  • Engineered biomechanical features including joint angles, angular velocities, motion vectors, and kinematic stress indicators.
  • Trained and evaluated Random Forest regression, K-Means clustering, and a deep neural network for stress prediction.
  • Improved dataset quality through preprocessing and scaling, increasing usable samples by nearly an order of magnitude.
  • Achieved significantly lower RMSE with the neural network compared to baseline models.
  • Validated predictions by correlating high stress scores with physically demanding movements.
